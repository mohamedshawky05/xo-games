XO is a fun, classic game that challenges you and encourages quick thinking. It was developed to run directly within the browser using modern web technologies such as HTML, CSS, and JavaScript.

This game aims to provide a light, entertaining experience suitable for all ages. It is designed with a simple, smooth interface that is compatible with all devices.

This game was developed by Mohamed Shawky, a front-end developer, as an open-source educational project to teach beginners the basics of building interactive games using JavaScript.
